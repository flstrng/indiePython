02_01_A
Напишите программу, которая выводит фразу «Я стану крутым программистом!»
три раза на отдельных строках; строки должны быть именно такие, символ в символ!.
Для удобства советую поместить эту фразу в переменную и затем использовать ее при выводе.
Sample Output:
Я стану крутым программистом!
Я стану крутым программистом!
Я стану крутым программистом!

02_01_B
Напишите программу, которая напечатает строку, состоящую из 777 букв "W"

02_01_C
Напишите программу, которая выводит «Лев Николаевич Толстой написал "Война и мир"» (без внешних кавычек).

02_01_D
Напишите программу, которая сначала считывает две фразы по очереди,
а потом воспроизводит их в той же последовательности, каждую на отдельной строчке.

Sample Input 1:
Hello!
Hi!
Sample Output 1:
Hello!
Hi!

02_01_E
Напишите программу, которая сначала считывает три фразы по очереди,
а потом воспроизводит их в обратной последовательности, каждую на отдельной строчке.

Sample Input:
Привет!
Артем
Как дела?
Sample Output:
Как дела?
Артем
Привет!

02_01_F
Напишите программу, которая считывает с клавиатуры слово (или фразу, неважно),
после чего выводит это же слово четыре раза на одной строке, разделяя их пробелами.

Sample Input 1:
ХА
Sample Output 1:
ХА ХА ХА ХА

02_01_G
Напишите программу, которая считывает с клавиатуры фразу и
выводит на экран количество символов, которое содержалось в ней,
учитывая все знаки препинания и пунктуации.

02_01_H
Напишите программу, которая считывает с клавиатуры два слова
(или фразы, неважно), после чего вы должны при помощи операции
конкатенации сцепить вторую фразу с первой и вывести результат на экран.

02_01_I
Напишите программу, которая считывает с клавиатуры слово (или фразу, неважно),
после чего вывести данную строку, увеличенную по длине  в 3 раза.

Sample Input 1:
Hello
Sample Output 1:
HelloHelloHello

02_01_J
Программа принимает на вход три символа через пробел в одну строку.
Необходимо вывести код каждого символа при помощи функции ord в определенном формате.

Sample Input 1:
A , y
Sample Output 1:
Simvol code A is 65.
Simvol code , is 44.
Simvol code y is 121.
